Calibration: Place a reference object of known size (e.g. a 50 mm × 50 mm gauge block) in the camera frame alongside your part. Count its pixel width from the source canvas, then set mm/px = known_mm ÷ pixel_width. For best accuracy use a backlit lightbox and fix the camera position with a copy stand.
What it does
Live video input
Connect any USB or built-in camera. The scanner runs in real time so you can adjust lighting and positioning before capturing.
Contour & hole detection
Gaussian blur, adaptive threshold, and Douglas-Peucker simplification extract the outer profile plus all interior holes — for any flat shape.
mm-accurate measurement
Set the mm/px ratio once using a reference gauge and every subsequent dimension — width, height, hole diameter — outputs in real millimeters.
G-code export
Generates standard RS-274 G-code with G0/G1 linear moves for the profile and G2/G3 arcs for holes. Feed rate, cut depth, and safe height are all configurable.
Runs in your browser
Zero installation. All processing happens client-side — no server, no uploads, no subscription. Works offline once the page is loaded.
Any flat shape
Cross brackets, corner gussets, plates, flanges — if it's flat and has contrast against a background, MetalScan will trace it.
